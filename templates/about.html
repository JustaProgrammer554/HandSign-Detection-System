About the project

Explanation

Our project is built for real-time hand sign recognition and imitation for our robotic hand. It can be used for many applications in different areas.

Features

  • Real-time hand sign detection.
  • Robotic hand imitate hand signs.

Components

  • Raspberry pi 5 8GB RAM
  • Arduino Mega
  • Transceiver
  • Hand Skeleton with Servo Motors
  • Battery
  • Camera
  • Speaker

We use websockets and uart for communication.
